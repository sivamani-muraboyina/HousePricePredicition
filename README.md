# California House Price Predictor 🏠

A machine learning web application that predicts median house prices in California using a Ridge Regression model.

## 🚀 Features
* **Machine Learning:** Built with Scikit-Learn using the California Housing dataset.
* **Interactive UI:** Built with Streamlit to allow real-time price estimation.
* **Comparison:** Evaluated using Linear, Ridge, and Lasso regression techniques.

## 🛠️ Tech Stack
* **Language:** Python 3.12
* **Libraries:** Pandas, Scikit-Learn, Streamlit, Joblib
* **Model:** Ridge Regression ($R^2 \approx 0.57$)

## 📦 Installation
1. Clone the repo: `git clone <your-repo-link>`
2. Create virtual environment: `python -m venv venv`
3. Install dependencies: `pip install -r requirements.txt`
4. Run the app: `streamlit run app.py`
